== Description ==

This plugin creates a glossary for your WordPress powered site. 
It will create an additional section in the admin panel, where you can add new terms to your glossary

A few notes about the sections above:

*   "Browse glossary" here you will see all terms.
*   "Add term" here you will add new term.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `PdAlex-Glossary` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert `{pdalex_glossary}` on the page mode html where you want to see a glossary

For more information, including installation and usage instructions, please visit the page for this plugin:

[PdAlex Glossary](http://pdalex.org.ua/?page_id=254)
